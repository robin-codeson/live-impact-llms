# Wat is het verschil tussen een project kick-off en een reguliere kick-off?

Een reguliere kick-off markeert een nieuw jaar of nieuwe periode. Een project kick-off start een specifiek project met een afgebakend doel, team en deadline. De inhoud is concreter: scope, rollen, planning en risico's staan centraal. De energie blijft belangrijk, maar het draait om commitment op het werk dat eraan komt.
