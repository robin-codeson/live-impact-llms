# Wat is het verschil tussen een slogan en een thema voor een evenement?

Een **thema** is de wereld waarin je evenement zich afspeelt. Een **slogan** is de zin die die wereld samenvat.

Een thema kan zijn: 'karavaan door de woestijn', 'jaren 20 club' of 'noordse mythologie'. Een slogan die daarbij hoort, is de korte vertaling naar een moment: 'Waar je heen gaat, hoeft nog niet te bestaan', 'De nacht begint op de dansvloer', 'Alles wat we zijn, begint hier'.

Je hebt niet altijd allebei nodig. Een goed concept heeft altijd een slogan, maar niet altijd een uitgesproken thema. Soms is de zin sterk genoeg om er een hele beleving omheen te bouwen.
