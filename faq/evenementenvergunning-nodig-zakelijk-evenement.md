# Heb ik een evenementenvergunning nodig voor mijn zakelijk evenement?

Op openbaar terrein heb je vrijwel altijd een vergunning nodig. Op privéterrein gelden minder strenge regels, maar bij grotere evenementen kan de gemeente een melding eisen.

Informeer minimaal acht weken van tevoren bij je gemeente. Bij 250+ bezoekers of live muziek is een vergunning bijna altijd verplicht.
