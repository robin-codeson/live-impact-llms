# Wat zijn de voordelen van een evenement op eigen locatie?

Een evenement op eigen locatie biedt meerdere voordelen. Je betaalt geen huurkosten. Via de [WKR](/blog/werkkostenregeling-evenement-wkr) profiteer je van een fiscale vrijstelling. De sfeer is authentiek: gasten zien je bedrijf. Je hebt maximale controle over de ruimte en een unieke sfeer die je nergens anders vindt.
