# Hoe ver van tevoren moet je een bedrijfsuitje plannen?

Vier tot zes weken is een realistisch minimum. Populaire activiteiten zoals escape rooms of bijzondere restaurants zijn op gewilde data snel volgeboekt.

In populaire periodes (voor kerst, vlak voor de zomervakantie) reken je op 8-10 weken. Doe eerst een snelle peiling bij het team, zodat je meteen de beste optie kunt boeken.
